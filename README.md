I have started learning web development. This is my first javascript based mini-project. 
I have implemented a web based tic tac toe game using html, css, javascript. 
I have used concepts like DOM manipulation, event handling, arrays methods, callbacks using arrow functions and much more.
In this project, you can reset the game mid-way or start a new game after game ends. 
The game checks for a winner after every move and also detects if a draw takes place.
It is a complete mini-project, that handles inputs and shows proper outputs.
